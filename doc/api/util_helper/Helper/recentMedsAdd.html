void recentMedsAdd (Med m)

Adds a new medicament med to the globals.recentMeds list as the latest entry. Removes duplicates.

Implementation

static void recentMedsAdd(Med m) {
  globals.recentMeds.removeWhere((item) => item.pzn == m.pzn);
  m.isHistory = true;
  globals.recentMeds.add(m);
}
