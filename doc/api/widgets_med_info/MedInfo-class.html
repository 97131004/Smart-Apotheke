MedInfo class

Page that shows medicament information from the package leaflet. Input parameter is a medicament med. Loads the medicament information with a GET-Request from beipackzettel.de, then parses and displays it here. You can jump (autoscroll) to certain categories by clicking on the links at the top. The user can increase and decrease the text size.
